Ricing Adventures

Hello and welcome to my tales of how I riced my unix box!
Also usually reffered to as how I spent days tweaking config files until I got frustrated!

"Rice" is a word that is commonly used to refer to making visual improvements and customizations on one's desktop.

My Setup

I am basing my setup in ubuntu for several reasons. First of all I was not sure if I would enjoy my experience with no mouse use at all. Then, it was a quick setup. And finally I really like my apt-get :)

My window manager is i3-gaps. It is a fork of i3 which has extra features such as "gaps". Gaps are basically spaces between windows.

I'm also using an application launcher called rofi. It is very customizable and exactly what I was looking for.

For the code editor I am basically using neovim. Which I have to admit it was a bit of pain to get used to and setup, but now I am in LOVE with it! I now want vim shorcuts everywhere haha

I have also set up other editors like DOOM emacs (which is a fork of emacs); but in the end I basically stick to vim

My browser of choice is Firefox. It is basically what I am used to and allows a lot of customization with css and extensions.

I also wanted a bar so I am using polybar. It has a lot of community and it is very powerfull.
